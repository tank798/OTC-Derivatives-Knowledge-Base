# 手动补件入口

把自动爬虫暂时拿不到的官方 PDF、Word、WPS、HTML 保存件放到这里。建议按来源分目录：

- `dce/`: 大连商品交易所规则。
- `gfex/`: 广州期货交易所规则。
- `czce/`: 郑州商品交易所剩余规则。
- `csrc/`: 证监会 NERIS 页面提示“详见附件”的附件原文。
- `bse/`: 北交所转载上位法 PDF 附件；主库已有上位法正文，此目录优先级较低。

命名建议：

```text
来源_文件名_发布日期_官方URL片段.pdf
```

例如：

```text
dce_大连商品交易所交易规则_2024-xx-xx.pdf
gfex_广州期货交易所交易规则_2024-xx-xx.docx
csrc_衍生品交易业务风险揭示书参考范本_中期协字2025-25.pdf
```

如果网页不能下载附件，也可以在浏览器里“打印/另存为 PDF”后放进对应目录。
