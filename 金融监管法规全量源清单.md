# 金融监管法规全量源清单（第一轮）

更新时间：2026-07-07

目标：为“场外衍生品 / 股票 / 固收 / 基金 / 资管 / 跨境 / 银行间”相关产品合规知识库建立官方法规源地图。本文先解决“去哪抓、抓什么、先抓什么”，后续再逐条落库、解析条款、标注效力状态。

## 一、抓取原则

1. 只以官方源、监管机构源、自律组织源、交易场所源、市场基础设施源为主数据源。
2. 每份文件必须保留原始 URL、发布机构、发布日期、施行日期、文号、效力状态、附件 PDF/Word 原文。
3. 法规正文按章、节、条、款、项切分，不按固定字数切分。
4. 法规问答和产品审查答案必须引用到具体法规、条文或官方问答原文。
5. 征求意见稿、政策解读、新闻稿、培训材料可以入库，但不能与正式有效规则同权重。

## 七、第一轮缺口

1. 国家金融监督管理总局官网栏目需要进一步定位可稳定抓取的政策法规列表。
2. 上交所、北交所、中金所部分页面直接 curl 有 403 或 TLS 问题，需浏览器抓取或设置合规 UA。
3. 中证登、中债登、上海清算所的规则入口需要补充精确栏目 URL。
4. 证券业协会场外衍生品相关自律规则需要按关键词二次检索和人工核验。
5. 需要建立“废止/修订/新旧规则替代关系”表，否则问答会误引旧规。
